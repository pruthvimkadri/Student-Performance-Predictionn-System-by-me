# Student Performance Prediction System

## Overview
This project predicts student final grades using machine learning.

## Features
- Predicts student performance
- Flask web application
- Interactive UI
- Random Forest Classifier
- Real-time prediction

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

## Dataset Features
- Study Hours
- Attendance
- Internet Access
- Travel Time
- Math Score
- Science Score
- English Score

## Model Used
Random Forest Classifier
